# 表格
<!-- 例子一 -->
|123|234|345|
|:-|:-:|-:|
|abc|bcd|cde|
|abc|bcd|cde|
|abc|bcd|cde|
---
```
第一种写法
|123|234|345|
|:-|:-:|-:|
|abc|bcd|cde|
|abc|bcd|cde|
|abc|bcd|cde|
第二种写法
|123|234|345|
|:---|:---:|---:|
|abc|bcd|cde|
|abc|bcd|cde|
|abc|bcd|cde|
第三种写法
123|234|345
:-|:-:|-:
abc|bcd|cde
abc|bcd|cde
abc|bcd|cde
```

1. 自带格式化/排版信息，且规则简单友好。
2. 规则直观且确定，不用像office一样调整样式。
3. 对比HTML：更专注于内容，标签对内容的侵入性低。
4. 最大优点：效率！写md几乎就是写自动排版的纯文本。
